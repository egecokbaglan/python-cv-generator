# Python CV Generator

This project is a simple command-line based CV generator built with Python.

The goal of this project is to practice:
- User input handling
- String formatting
- File writing
- Basic program structure
- Clean GitHub project development

The program will collect user information and generate a formatted CV file automatically.

## Project Status
Currently under development.

## Planned Features
- Collect personal information
- Add education and skills sections
- Save CV as a text file
- Improved formatting
- Possible .docx export (future update)

## Technologies
- Python
